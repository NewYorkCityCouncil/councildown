# Wrapper for addPolygons

All the same inputs apply as the leaflet::addPolygons function, we just
use this wrapper to define the "defaults" for certain inputs

## Usage

``` r
addPolygons(map, smoothFactor = 0, weight = 0, ...)
```

## Arguments

- map:

  A `leaflet` map

## Value

A `leaflet` map with polygons added
